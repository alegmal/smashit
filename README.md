# SmashIt 👶

Baby keyboard smashing playground. Goes fullscreen, captures all keypresses, shows big colorful bursts. Press ESC to exit.

## Getting Started

```bash
npm install
npm run dev   # http://localhost:3001
```

## How it works

1. Click **START** → fullscreen + keyboard capture begins
2. Every key → big colorful label in the center + floating burst
3. **ESC** → exit fullscreen, return to landing

A safety timer stops capture automatically after 15 seconds.
